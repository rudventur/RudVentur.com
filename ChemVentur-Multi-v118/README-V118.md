## 🐛 TROUBLESHOOTING

### **AI Bots Not Spawning?**
- Make sure you're the first player to connect
- Check browser console for errors
- Refresh and try again

### **Chat Not Showing?**
- Ensure multiplayer is connected
- Check Firebase rules (must allow read/write)
- Look for chat window in bottom-right corner

### **Messages Not Sending?**
- Check your internet connection
- Verify Firebase database is online
- Try refreshing the page

### **Bots Acting Weird?**
- That's Chaos Bot! 😄
- If it's a real issue, check console logs
- Report bugs via GitHub issues
