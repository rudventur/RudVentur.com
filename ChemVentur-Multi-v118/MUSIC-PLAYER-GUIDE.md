### **Features:**
- ✅ Full video player embedded
- ✅ Auto-loop enabled (music repeats!)
- ✅ Volume control works
- ✅ Plays in background while you game
- ✅ Can minimize music window

### **Best For:**

## 🌐 DIRECT URL MODE

### **How to Use:**
1. Click **"Direct URL"** button
2. Paste a direct link to an audio file
3. Click **"▶️ Play from URL"**
4. Music streams from the web!

### **What You Can Paste:**
- Direct MP3: `https://example.com/song.mp3`
- Hosted audio: `https://mysite.com/music/track.ogg`
- CDN links: `https://cdn.example.com/audio.wav`

### **URL Requirements:**
- ⚠️ Must be a **direct link** to audio file
- ⚠️ Must end in `.mp3`, `.ogg`, `.wav`, etc.
- ⚠️ No playlists or streaming sites

### **Features:**
- ✅ Stream from anywhere
- ✅ Works with any public audio URL
- ✅ Auto-loop enabled
- ✅ Volume control

### **Best For:**
- 🎵 Self-hosted music
- 🎵 Game OST sites

## ⚠️ TROUBLESHOOTING

### **YouTube Not Playing?**
- ✅ Check internet connection
- ✅ Verify URL is correct YouTube link
- ✅ Try just the video ID instead
- ✅ Some videos may be age-restricted
- ✅ Refresh page and try again

### **SoundCloud Not Loading?**
- ✅ Make sure it's a direct track URL
- ✅ Playlist URLs might not work
- ✅ Private tracks won't play
- ✅ Try a different track

### **Local File Won't Play?**
- ✅ Check file format (MP3 usually best)
- ✅ File might be corrupted
- ✅ Try converting to MP3
- ✅ Make sure file isn't too large (>100MB)

### **Direct URL Fails?**
- ✅ Must be direct link to audio file
- ✅ Can't use streaming sites
- ✅ Check URL spelling
- ✅ Server might block external access

### **No Sound?**
- ✅ Check volume slider in music player
- ✅ Check browser volume
- ✅ Check system volume
- ✅ Make sure Play button was clicked

### **Player Window Missing?**
- ✅ Click music button again to reopen
- ✅ Might be behind other windows
- ✅ Refresh page if completely lost
